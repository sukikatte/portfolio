Digital Mahjong

Java, JavaFX, Event-driven UI GitHub →

Focus: Core game logic, win detection, AI opponents, and event-driven UI with JavaFX.

Learning: Software is not only about algorithms — user experience is what makes technology meaningful.

Turning Point: Professor's feedback ("lacked user thinking") reshaped how I think about software design.

Guess Songs

Java, Android Studio, Firebase, Cloud OSS GitHub →

Focus: Native Android app with modular MVVM architecture, Classic/Timed/Multiplayer modes.

Learning: Engineering can create experiences — such as real-time competition and responsive UI — not just functions.

Impact: Realized that architecture and interface must evolve together for engaging products.

Dwen Dwen's Neighbor

Python, Flask, Firebase, JavaScript, ECharts GitHub →

Focus: Modular, bilingual platform for multi-user interaction with animal encyclopedia and virtual panda care.

Learning: Designing experiences requires merging engineering with storytelling, interaction, and accessibility.

Result: 3rd Prize, China Innovation & Entrepreneurship Competition (BJUT).

Chinese Medicine Platform

Vue.js, JavaScript, Flask, SQLAlchemy, Fuzzy Matching GitHub →

Focus: Bilingual full-stack platform with role-based access control and AI consultation chat.

Learning: Building platforms that solve real-world problems requires understanding user needs deeply.

Vision: Moving from "product developer" to "experience designer" shaping meaningful digital ecosystems.

Parrot Ordering System

Python, Flask, JavaScript, HTML, CSS GitHub →

Focus: Modular food ordering system with role-based workflows (Admin/Cook/Customer).

Learning: Interactive features like real-time menu management and social review systems enhance user engagement.

Impact: The ultimate goal is not technology itself, but the value it creates for people and society.

Event Management Platform

Spring Boot, MySQL, Docker, Maven GitHub →

Focus: Full-stack event management platform with multi-role access control (Admin/Organizer/User).

Learning: Secure backend with Spring Security, containerized deployment via Docker, complete SDLC with 70+ UML diagrams.

Result: Comprehensive documentation and scalable architecture for enterprise-level applications.

Frankenstories: Story Generation from Reusable Parts

LLM, Multimodal AI, ComiXML, Computational Creativity GitHub →

Focus: Developing an LLM-powered story generation system that recombines visual and textual elements from existing comic strips (ComiXML) to produce new narratives, aiming to explore computational creativity and automated storytelling.

Learning: Building a multimodal content generation pipeline, integrating language models with structured visual data for creative narrative production.

Impact: Final Year Project under Assoc. Prof. Tony Veale, exploring the intersection of AI and creative storytelling (Sept. 2025 - Present).

About Me

The Designer's Mindset

I'm a software engineer with a passion for creating technology that truly resonates with people. My journey began with functional programming, but evolved into a deep exploration of how systems can be designed to serve human needs.

"Years of playing music taught me how timing shapes emotion — a principle I now apply to designing intuitive user flows. Drawing trained my eye for visual balance, which guides my interface design decisions."

功能为基、体验为魂 — Functionality is the foundation, experience is the soul.
